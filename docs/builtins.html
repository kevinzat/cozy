Default Definitions

When the Definitions field is left empty, Cozy loads the following integer definitions. These can be used with the defof and undef rules.

Even

forall x, Even(x) <-> (exists y, x = 2*y)

A number is even if it equals twice some integer.

Odd

forall x, Odd(x) <-> (exists z, x = 2*z + 1)

A number is odd if it equals twice some integer plus one.

Divides

forall a, forall b, Divides(a, b) <-> (exists k, b = k*a)

Divides(a, b) means a divides b (i.e., b is a multiple of a).

Congruent

forall a, forall b, forall m, Congruent(a, b, m) <-> Divides(m, a - b)

Congruent(a, b, m) means a is congruent to b modulo m.

Prime

forall p, Prime(p) <-> not (p = 1) and (forall x, Divides(x, p) -> (x = 1) or (x = p))

A number is prime if it is not 1 and its only divisors are 1 and itself.

Default Theorems

When the Theorems field is left empty, Cozy loads the following theorems. These can be referenced with the cite and apply rules.

Arithmetic Theorems

NameStatement
MultEqns forall a, forall b, forall c, forall d, (a = b and c = d) -> (a*c = b*d)
DivEquation forall a, forall b, forall c, (c*a = c*b and not (c = 0)) -> (a = b)

MultEqns lets you multiply both sides of two equations together. DivEquation lets you cancel a nonzero common factor.

Division Algorithm

NameStatement
Division forall a, forall b, (0 < b) -> (a = div(a,b)*b + mod(a,b) and 0 <= mod(a,b) and mod(a,b) < b)
DivisionUnique forall a, forall b, forall q, forall r, (a = q*b + r and 0 <= r and r < b) -> (q = div(a,b) and r = mod(a,b))

The division algorithm gives quotient div(a,b) and remainder mod(a,b) for any integers with positive divisor. Uniqueness says these are the only quotient and remainder.

GCD Theorems

NameStatement
GCD forall a, forall b, (0 < a and 0 <= b) -> (Divides(gcd(a,b),a) and Divides(gcd(a,b),b) and (forall d, Divides(d,a) and Divides(d,b) -> (d <= gcd(a,b))))
GCDUnique forall a, forall b, forall x, (Divides(x,a) and Divides(x,b) and (forall d, Divides(d,a) and Divides(d,b) -> (d <= x))) -> x = gcd(a,b)

The GCD theorems establish that gcd(a,b) is the greatest common divisor: it divides both a and b, and is the largest such divisor.

Divisibility Theorems

NameStatement
DividesOne forall a, Divides(a, 1) -> a = 1

Set Definitions

The following set definitions are always available (they cannot be overridden). They are used internally by the set-related rules.

OperationDefinition
Complement (~A) forall a, forall x, x in ~a <-> not (x in a)
Union (A cup B) forall a, forall b, forall x, x in (a cup b) <-> (x in a or x in b)
Intersection (A cap B) forall a, forall b, forall x, x in (a cap b) <-> (x in a and x in b)
Difference (A \ B) forall a, forall b, forall x, x in (a \ b) <-> (x in a and not (x in b))
Subset (A subset B) forall a, forall b, a subset b <-> (forall x, x in a -> x in b)
Set Equality (A sameset B) forall a, forall b, a sameset b <-> (a subset b and b subset a)
Note Set definitions are built into Cozy's internals and cannot be parsed from text (because they use higher-order set variables). They are always loaded alongside whatever custom definitions you provide.
